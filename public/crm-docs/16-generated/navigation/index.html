    Navigace (generováno)¶

    Automatický přehled položek CRM menu (GET routy s prefixem /crm). Popisy lze doplnit v docs/_meta/menu.php.

    Dashboard¶

    Výchozí přehled (widgety, stav synchronizací, klíčové KPI).

    Cesta Route name Popis
    / crm.dashboard Hlavní přehled a rychlé akce.

    Firmy¶

    Evidence firem (profil, revenue, průmysl) – vstup do detailu vztahů.

    Cesta Route name Popis
    /companies companies.index (popis chybí)
    /companies/create companies.create (popis chybí)
    /companies/{company} companies.show (popis chybí)
    /companies/{company}/edit companies.edit (popis chybí)
